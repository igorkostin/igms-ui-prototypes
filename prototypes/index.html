iGMS hero animation prototypes

Sandbox of background-animation experiments for the new iGMS landing hero. All effects generated in-browser (no video, no heavy assets), tunable, brand-themable.

  • canvas

    hero01 — copperx waves

    Flow-field of soft horizontal wave lines on canvas. Tunable density, hue range, speed. Inspired by copperx.io.
  • css

    hero02 — castle blobs

    Pastel radial-gradient blobs drifting in the background. Pure CSS animation. Inspired by castle.io.
  • svg

    dot-ring — Salesloft-style

    Rotating ring / horizon arc of dots with density falloff. Inline SVG + CSS rotate. Inspired by salesloft.com.
  • image

    hero03 — trunk mountains

    Sketch-drawn ink mountains under the hero, faded into the white background. Static, no animation. Inspired by trunk.io.
  • css + js

    hero04 — refold rails

    Animated color-bar rails on both sides of the hero. Bars pulse with a staggered wave through a brand palette. Inspired by refold.ai.
Source on GitHub: igorkostin/igms-ui-prototypes. Deploy via ./deploy.sh (rsync to /var/www/prototypes/).
